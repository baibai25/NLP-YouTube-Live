Natural language processing for YouTube Live

[https://qiita.com/baibai25/items/311041affde6749e7161](https://qiita.com/baibai25/items/311041affde6749e7161)

## Usage
1. Download YouTube Live comments
- only supports archived stream video
```
$ python get_comment.py 'video url'
```

2. Analyze chat comments data
- Start `jupyter lab` or `jupyter notebook`
- Open `nlp.ipynb` and run
- `nlp.ipynb` only supports Japanese.
- `nlp_multilingual.ipynb` supports Japanese, English and Chinese.
